144.二叉树的前序遍历

题目

二叉树的前序遍历

解题思路一：基于递归

  1. 定义一个函数，当root为空时则退出
  2. 而root不为空时，则将其值增加到res中，并且继续对root的左右孩子执行该函数
  3. 最终返回res，代码如下
void fun(vector<int> &res, TreeNode* root){
    if(root==nullptr){
        return;
    }
    res.push_back(root->val);
    fun(res,root->left);
    fun(res,root->right);
}
vector<int> preorderTraversal(TreeNode* root) {
    vector<int> res;
    fun(res,root);
    return res;
}

解题思路二：基于迭代

  1. 定义一个栈用来存放节点
  2. 首先，将根节点压入栈
  3. 对栈执行如下操作，若栈不为空，则将栈顶点弹出，并将栈顶的值加入到res中，如果栈顶存在右孩子，则将右孩子入栈；如果栈顶存在左孩子，则将左孩子入栈
  4. 最终返回res，代码如下：
vector<int> preorderTraversal(TreeNode* root) {
    vector<int> res;
    if(!root){return res;}
    stack<TreeNode*> stk;
    stk.push(root);
    while(!stk.empty()){
        TreeNode* tmp=stk.top();
        stk.pop();
        res.push_back(tmp->val);
        if(tmp->right){stk.push(tmp->right);}
        if(tmp->left){stk.push(tmp->left);}
    }
    return res;
}

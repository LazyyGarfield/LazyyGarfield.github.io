毕业旅行问题

题目

毕业旅行问题

小明目前在做一份毕业旅行的规划。打算从北京出发，分别去若干个城市，然后再回到北京，每个城市之间均乘坐高铁，且每个城市只去一次。由于经费有限，希望能够通过合理的路线安排尽可能的省一些路上的花销。给定一组城市和每对城市之间的火车票的价钱，找到每个城市只访问一次并返回起点的最小车费花销。

描述

输入描述

  1. 第一行输入城市个数n
  2. 接下来输入n行，每一行输入n个数字，为城市间的车票价钱

输出描述

  1. 输出最小车费花销s

示例

输入

4
0 2 6 5
2 0 4 4
6 4 0 2
5 4 2 0

输出

13

解答

实际上，该问题是经典的旅行商TSP问题，具体解决方法有回溯法和动态规划算法

回溯法

  1. 定义memory数组，记录是否遍历该城市
  2. 如果没有遍历过该城市，则到达该城市，并从该城市继续向下遍历
  3. 需要注意的是，采用回溯法时，回溯时需要取消该城市的遍历
  4. 由于回溯法时间复杂度较高，需要剪枝，那么则判断到达城市后的当前花费是否大于result，如果已经超过result，则剪掉该分支
  5. 代码如下
#include <iostream>
#include <vector>
using namespace std;

void fun(vector<vector<int>> nums,int &result,vector<int> memory,int n,int row,int num,int sum_memory){
    if(num>=result){
        return;
    }
    if(sum_memory==n){
        result=min(result,num+nums[row][0]);
        return;
    }
    for(int i=1;i<n;i++){
        if(memory[i]==0){
            memory[i]=1;
            fun(nums,result,memory,n,i,num+nums[row][i],sum_memory+1);
            memory[i]=0;
        }
    }
}

int main() {
    // 初始化
    int n;
    cin>>n;
    vector<vector<int>> nums(n,vector<int>(n));
    for(int i=0;i<n;i++){
        for(int j=0;j<n;j++){
            cin>>nums[i][j];
        }
    }
    // 计算结果
    vector<int> memory(n,0);
    memory[0]=1;
    int result=1000;
    fun(nums,result,memory,n,0,0,1);
    cout<<result<<endl;
    return 0;
}
// 64 位输出请用 printf("%lld")

但是，最终该代码的时间复杂度仍较高

动态规划

  1. 定义一个dp数组，其中dp[i][j]的i是一个二进制形式的数，表示经过城市的集合，例如0111则表示经过了城市012，dp[i][j]表示经过了i中的城市，并且以j结尾的路径长度
  2. 首先，该dp数组中的值全为MAX，而dp[1][0]为0，这是因为从城市0出发到达城市0的路径长度为0
  3. dp的更新为dp[i | (1 << k)][k] = min(dp[i | (1 << k)][k],dp[i][j] + nums[j][k])
  4. 最终，我们可以得到从所有城市返回城市0的路径长度，从中选择最小的路径长度即可
  5. 代码如下
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
int getAns(vector<vector<int>> &nums){
    const int MAX = 0x0fffffff;
    int n = nums.size();
    int stateNum = 1 << n;//即2的n次方，也就是第n个城市
    // dp[i][j]中的i是一个二进制形式的数，表示经过城市的集合，如0111表示经过了城市0,1,2    
    // dp[i][j]表示经过了i中的城市，并且以j结尾的路径长度
    vector<vector<int> > dp(stateNum,vector<int>(n,MAX));
    dp[1][0] = 0; //从城市0出发，所以经过城市0，以城市0结尾的路径为0
    //从城市0出发，更新和其他城市的距离
    for(int i=1;i<stateNum;i++){//最大值为2的n次方
        for(int j=0;j<n;j++){//选择下一个加入到集合的城市j
            if(dp[i][j] != MAX){ //如果已经访问过，再继续走
                for(int k=0;k<n;k++){//尝试所有其他节点k
                    if( (i & (1 << k) ) == 0){ 
                        //没有访问过k，且从这里到k的距离小于原来的距离，则更新
                        dp[i | (1 << k)][k] = min(dp[i | (1 << k)][k],dp[i][j] + nums[j][k]);
                       
                    }
                }
            }
        }
    }
    int res = MAX;
    for(int i=1;i<n;i++){
        res = min(res,dp[stateNum-1][i] + nums[i][0]);
    }//得到经过了所有城市，并以i结尾的最小路径长度。
    return res;
}
int main(){
   int n;
    while(cin>>n){
        vector<vector<int>> edges(n,vector<int>(n,0));
        int x;
        for(int i=0; i<n; i++){
            for(int j=0; j<n; j++){
                cin>>edges[i][j];
            }
        }
        cout<<getAns(edges)<<endl;
    }
    return 0;
}

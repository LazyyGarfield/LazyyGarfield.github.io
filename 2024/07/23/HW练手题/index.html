第1题：公共片段

题目

获取两个数组的公共片段，值相等的连续片段即可，以数组的形式返回。如果没有公共片段，则返回-1

每个数组的长度不超过20个元素，且不会为空

示例

输入：【1 2 2 3 9 1 5】和【9 2 2 3 6 8】

输出：【2 2 3】

思路

  1. 因为数据非常小，因此直接采用暴力枚举
  2. 将第一个数组的所有子数组添加到哈希表中，然后检索第二个数组的所有子数组是否在哈希表中，并且更新最长公共部分
  3. 代码如下：
vector<int> fun(vector<int> v1,vector<int> v2){
    // 两个数组的长度
    int n1=v1.size(),n2=v2.size();
    // 将第一个数组的所有子数组加入到哈希表中
    unordered_set<vector<int>> s1;
    for(int i=0;i<n1;i++){
        vector<int> tmp;
        for(int j=i;j<n1;j++){
            tmp.push_back(v1[j]);
            s1.insert(tmp);
        }
    }
    // 定义公共部分数组
    vector<int> res;
    // 判断第二个数组的子数组是否存在在哈希表中
    for(int i=0;i<n2;i++){
        vector<int> tmp;
        for(int j=i;j<n2;j++){
            tmp.push_back(v2[j]);
            if(s1.count(tmp)!=0&&tmp.size()>res.size()){
                res=tmp;
            }
        }
    }
    // 判断res是否为空
    return res.empty()?-1:res;
}

第2题：采矿

题目

露天矿采矿作业的特点是规模大，矿石和废料的移动量达到百万吨，运输成本开销较大，需要寻求一种最优的运输路径节省成本。

已知矿场可以划分成N*M的网格图，每个网格存在地形的差异，因此通过不同网格时，成本开销存在差异。

网格有以下5种类型：

  1. 标志为’S’的网格为运输起点；
  2. 标志为’E”的网格时运输终点；
  3. 标志为’B’的网格为阻塞点，不允许通行；
  4. 标志为’C’的网格为检查点，矿车在运输路径中，至少需要进入一次检查点。
  5. 标志为‘数字”的网格，其数字表示经过该网格的成本开销。

运输矿车只能上下左右4个方向运行，不允许斜对角进入其他网格。必要时可重复进入网格。
请根据输入的网格图，寻求一条从S网格到E网格，并且至少经过一次检查点的最低成本运输路径，并输出其成本开销。

示例

输入：N*M的网格图【S 4 5;7 B 3;C 9 E】

输出：16

思路

  1. 基于逆向思维，从C点出发，找到C->S和C->E的最短路径
  2. 遍历地图的所有C点，找到二者和的最小值即可

解答

  1. 定义函数，找到从C点到地图上任意一个点的最短距离
vector<vector<int>> fun(vector<vector<string>> matrix,int c_i,int c_j){
    int n=matrix.size(),m=matrix[0].size();
    // 定义距离矩阵和记录矩阵
    vector<vector<int>> distance(n,vector<int>(m,INT_MAX));
    vector<vector<bool>> visited(n, vector<bool>(m, false));
    // 当前点的距离为0
    distance[c_i][c_j] = 0;
    // 定义一个堆，每个元素存放【距离，横坐标，纵坐标】
    priority_queue<tuple<int, int, int>, vector<tuple<int, int, int>>, greater<tuple<int, int, int>>> h;
    h.push(make_tuple(0, c_i, c_j));
    // 以一个堆存放即将扩展的网格，朝四个方向扩展，更新其四个方向的最短距离
    while (!h.empty()) {
        int d, i, j;
        tie(d, i, j) = h.top();
        h.pop();
        if (visited[i][j]) continue;
        visited[i][j] = true;
        
        // 朝四个方向扩展，更新四个方向的最短距离
        vector<pair<int, int>> directions = {{i+1, j}, {i, j+1}, {i-1, j}, {i, j-1}};
        for (auto [ni, nj] : directions) {
            if (ni < 0 || nj < 0 || ni >= n || nj >= m || visited[ni][nj] || matrix[ni][nj] == "B") continue;
            int w = 0;
            if (matrix[ni][nj] == "C" || matrix[ni][nj] == "E" || matrix[ni][nj] == "S") {
                w = 0;
            } else {
                w = stoi(matrix[ni][nj]);
            }
            if (distance[ni][nj] > distance[i][j] + w) {
                distance[ni][nj] = distance[i][j] + w;
                h.push(make_tuple(distance[ni][nj], ni, nj));
            }
        }
    }
    return distance;
}
  1. 遍历所有的C点，找到C到S和C到E的二者和最小点
int n=matrix.size(),m=matrix[0].size();
// 存放所有的C点
vector<vector<int>> C;
for(int i=0;i<n;i++){
    for(int j=0;j<m;j++){
        if(matrix[i][j]=="C"){
            C.push_back({i,j});
        }
    }
}

// 找到S点和E点
int res = INT_MAX;
int s_i, s_j, e_i, e_j;
for(int i=0;i<n;i++){
    for(int j=0;j<m;j++){
        if(matrix[i][j]=="S"){
            s_i=i;s_j=j;
        }
        else if(matrix[i][j]=="E"){
            e_i=i;e_j=j;
        }
    }
}

// 遍历所有的C点，找到最小的二者和
for(int i=0;i<C.size();i++){
    vector<vector<int>> dis = fun(matrix,C[i][0],C[i][1]);
    res = min(res,dis[s_i][s_j]+dis[e_i][e_j]);
}

// 返回res
return res==INT_MAX?-1:res;

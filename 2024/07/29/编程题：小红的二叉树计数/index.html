编程题：小红的二叉树计数

题目

小红的二叉树计数

小红定义一个二叉树为“好二叉树”，当且仅当该二叉树所有节点的孩子数量为偶数（0或者2）

小红想知道，【n个节点】组成的好二叉树，共有多少种不同的形态？答案请对【10^9+7】取模

示例1

输入

5

输出

2

说明

  1. 【5个节点】的”好二叉树”一共有2种，分别为【p,p,p,p,p】以及【p,p,p,null,null,p,p】

示例2

输入

7

输出

5

说明

  1. 【7个节点】的“好二叉树”一共有5种
  2. 左右均为3个节点的有一种
  3. 左边5个节点，右边1个节点的有两种，反过来也是2种
  4. 最终结果为5种

解答

  1. 可以基于动态规划思想，计算【第i个节点】的“好二叉树”数量时，可以将其分为左右子树的“好二叉树”数量之和
  2. 示例：【7个节点】的“好二叉树”数量可以分为【左子树1个节点，右子树5个节点】和【左子树3个节点，右子树3个节点】和【左子树5个节点，右子树1个节点】计算
  3. 需要注意的是，需要进行两次取模运算，并且需要将【10^9+7】提前定义为一个int变量
  4. 代码如下
class Solution {
public:
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * 
     * @param n int整型 
     * @return int整型
     */
    int cntOfTrees(int n) {
        int mod=(1e+9)+7;
        if(n%2==0){
            return 0;
        }
        // 1 3 5 7 9
        // 1 1 2 5 10+4
        vector<long long> dp(n+1);
        dp[1]=1;
        for(int i=3;i<=n;i=i+2){
            long long num=0;
            for(int j=1;j<i;j=j+2){
                num+=(dp[j]*dp[i-1-j])%mod;
                num%=mod;
            }
            dp[i]=num;
        }
        return dp[n];
    }
};

编程题：小美的平衡矩阵

题目

[小美的平衡矩阵](牛客网 - 找工作神器|笔试题库|面试经验|实习招聘内推，求职就业一站解决_牛客网 (nowcoder.com))

小美拿到了一个【n×n】的矩阵，其中每个元素是 0 或者 1。
小美认为一个矩形区域是完美的，当且仅当该区域内 0 的数量恰好等于 1 的数量。
现在，小美希望你回答有多少个【i×i】的完美矩形区域。你需要回答【1<=i<=n】的所有答案。

描述

输入描述：

  1. 第一行输入一个正整数n，代表矩阵大小
  2. 接下来的n行，每行输入一个【长度为n的01字符串】，用来表示矩阵。其中【1<=n<=200】

输出描述：

  1. 输出n行，第i行输出【i*i】的完美矩形区域的数量

示例

输入

4
1010
0101
1100
0011

输出

0
7
0
1

0表示【1×1】的完美矩形区域数量为0个，7表示【2×2】的完美矩形区域数量为7个

解答

  1. 首先，代码题为ACM模式，需要注意输入和输出的格式
  2. 进行初始化，第一行输入【n】
  3. 之后，每一行都输入一个字符串，并且对该字符串进行遍历
  4. 此时，定义一个前缀和数组prefix，其中prefix[i][j]表示【第i行从第0个符号到第j个符号的数字1的数量】。那么假如我们要计算【一个2×2矩形的1的数量】，我们就可以通过计算【矩形后边缘前缀和-矩形前边缘前缀和】得到
  5. 通过遍历【i×i】的矩形，判断其矩形是否为完美矩形区域，最终返回res
  6. 需要注意的是，res中奇数行的个数一定是0，因为该矩形区域的格子数量为奇数，不可能为完美矩形，不需要计算
  7. 完整代码如下
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
    // 初始化
    int n;
    cin>>n;
    // 定义数组prefix,prefix[i][j]表示第i行中从num[i][0]到num[i][j]中1的数量
    vector<vector<int>> prefix(n,vector<int>(n));
    for(int i=0;i<n;i++){
        string num;
        cin>>num;
        for(int j=0;j<n;j++){
            if(num[j]=='1'){
                if(j==0){prefix[i][j]=1;}
                else{prefix[i][j]=prefix[i][j-1]+1;}
            }
            else{
                if(j==0){prefix[i][j]=0;}
                else{prefix[i][j]=prefix[i][j-1];}
            }
        }
    }
    // 计算结果
    vector<int> result(n);
    for(int i=1;i<n;i++){
        if(i%2!=0){
            // 计算(i+1)*(i+1)的矩阵里1的个数
            int num=0;
            for(int col=i;col<n;col++){
                for(int row=0;row<n-i;row++){
                    int sum=0;
                    // 计算矩阵1的个数
                    for(int j=row;j<=row+i;j++){
                        if(col==i){sum+=prefix[j][col];}
                        else{sum+=prefix[j][col]-prefix[j][col-i-1];}
                    }
                    if(sum==(i+1)*(i+1)/2){num++;}
                }
            }
            result[i]=num;
        }
    }
    // 输出
    for(int i=0;i<n;i++){
        cout<<result[i]<<endl;
    }
    return 0;
}

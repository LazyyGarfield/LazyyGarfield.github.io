第1题

题目

示例

输入：1 8 9 5

输出：5 8 1 9

思路

  1. 为了便于查找序列的中位数，我们可以将序列处理为大根堆和小根堆
  2. 控制大根堆和小根堆的数字数量相差为1，并且大根堆存放比中位数小的序列，小根堆存放比中位数大的序列，此时中位数必定在两个堆的堆顶产生
  3. 当输出中位数时，我们只需要输出其中一个堆的堆顶即可，然后对两个堆进行调整

解答

  1. 首先，我们将序列处理为两个堆。遍历序列，每一个数先加入到大根堆
  2. 我们需要保持大根堆和小根堆之间的数量关系，有两种关系。第一种，两个堆数量相同；第二种，大根堆比小根堆多1个数字。
  3. 并且，我们需要保持大根堆和小根堆的堆顶元素关系，大根堆的堆顶元素必须小于小根堆的堆顶元素
  4. 因此，每插入一个元素，先比较两个堆顶的元素大小关系，再进行两个堆的数量调整
  5. 当形成两个堆之后，我们需要找到中位数。若此时两个堆数量不同，则中位数为大根堆堆顶；若此时两个堆数量相同，则中位数依然为大根堆堆顶。所以，中位数必然为大根堆堆顶，但弹出中位数之后，我们需要对两个堆进行数量调整
  6. 最终，循环输出中位数，代码如下：
#include <iostream>
#include <queue>
#include <vector>
#include <functional> // 用于std::greater和std::less

class MedianFinder {
private:
    std::priority_queue<int> maxHeap; // 用于存储较小的一半元素（最大堆）
    std::priority_queue<int, std::vector<int>, std::greater<int>> minHeap; // 用于存储较大的一半元素（最小堆）
    
    void balanceHeaps() {
        if (maxHeap.size() > minHeap.size() + 1) {
            minHeap.push(maxHeap.top());
            maxHeap.pop();
        } else if (minHeap.size() > maxHeap.size()) {
            maxHeap.push(minHeap.top());
            minHeap.pop();
        }
    }

public:
    void addNum(int num) {
        // 将新元素添加到适当的堆中
        maxHeap.push(num);
        if (maxHeap.top() > minHeap.top()) {
            // 交换两个堆的根，以保持中位数的正确性
            std::swap(maxHeap.top(), minHeap.top());
        }
        balanceHeaps();
    }

    double findMedian() {
        return maxHeap.top();
    }

    void printMedian() {
        while (!maxHeap.empty()) {
            std::cout << findMedian() << " ";
            // 从最大堆中删除中位数元素
            maxHeap.pop();
            balanceHeaps();
        }
    }
};

int main() {
    MedianFinder finder;
    std::vector<int> sequence = {9, 5, 7, 3, 1, 9, 8, 2};
    for (int num : sequence) {
        finder.addNum(num);
    }
    finder.printMedian();
    return 0;
}

第2题

题目

给定一个仅由小写字母组成、长度为n的字符串s，则字符串s有【2^n-1】个非空子序列。

请你求出所有子序列中不同字符的个数总和，由于答案较大，你需要输出对【10^9+7】取模后的结果。

示例

1）输入：abcde

输出：80

2）输入：aaaa

输出：15

思路

  1. 对于每个字符，计算字符c出现的数量，记为【cnt[c]】
  2. 从cnt[c]中至少取1个c出来有【2^cnt[c]-1】中可能，而从剩余【n-cnt[c]】个字符中取任意个字符有【2^(n-cnt[c])】中可能，两者相乘即为字符c的贡献
  3. 示例，在【abcde】中，【a】的数量为1，则取【a】有1种可能；而剩余为【4】个字符，则有【2^4=16】种方案，这16种方案与a组成子序列，则这些子序列中【a】的字符个数一共有【1×16=16】，分别对【5】个字符进行求解，最终为【80】
  4. 代码如下

解答

int fun(string s){
    int n=s.size();
    unordered_map<char,int> mp;
    for(int i=0;i<n;i++){
        mp[s[i]]++;
    }
    int res=0;
    for(int i=0;i<mp.size();i++){
        res+=(2^mp[i]-1)*(2^(n-mp[]));
        res%=(10^9+7);
    }
    return res;
}

第3题

题目

343.整数拆分

给定一个正整数 n ，将其拆分为 k 个 正整数 的和（ k >= 2 ），并使这些整数的乘积最大化。

返回 你可以获得的最大乘积 。

示例

输入：n=10

输出：36

解释：10=3+3+4，3×3×4=36。

解答：基于动态规划

  1. 对于每个数n，定义一个数组【dp[n+1]】，其中【dp[i]】存放以i拆分的整数最大乘积

  2. 【dp[i]】的更新是基于以下两种可能

    1）i拆分为j和i-j，且i-j不再拆分，则此时乘积为【j×(i-j)】

    2）i拆分为j和i-j，而i-j继续拆分，则此时乘积为【j×dp[i-j]】

  3. 因此，【dp[i]=j×max(i-j,dp[i-j])】，其中j的范围由【1到i-1】

  4. 代码如下：

int integerBreak(int n) {
    vector<int> dp(n+1);
    dp[1]=1;
    for (int i = 2; i <= n; i++) {
        for (int j = 1; j < i; j++) {
            dp[i]=max(dp[i], j*max(i-j,dp[i-j]);
        }
    }
    return dp[n];
}

时间复杂度为O(n^2)，空间复杂度为O(n)

优化

  1. 实际上，【dp[i]】的更新可以由以下决定：【dp[i]=max(2×(i-2),2×dp[i-2],3×(i-3),3×dp[i-3])】，具体由数学证明可得
  2. 因此，代码如下：
int integerBreak(int n) {
    vector<int> dp(n+1);
    dp[1]=1;dp[2]=1;
    for (int i = 3; i <= n; i++) {
        dp[i]=max(max(2*(i-2),2*dp[i-1]),max(3*(i-3),3*dp[i-3]));
    }
    return dp[n];
}

时间复杂度为O(n)，空间复杂度为O(n)

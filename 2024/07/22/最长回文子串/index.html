5.最长回文子串

题目

最长回文子串

解题思路：

  1. 对字符串 s 的每一个字符 i 进行遍历，以 i 为中心，向左右两侧扩展，找到以 i 为中心的最长回文子串 s1

  2. 同样，以 i 和 i+1 为中心进行拓展，找到最长回文子串 s2，返回 s1 和 s2 其中最长的回文子串，然后更新最长回文子串res

  3. isPalding() 函数用于找到以 left 和 right 为中心的最长回文子串。具体实现为，从 left 和 right 分别向左右两侧扩展，如果左右两侧字符相等，则继续扩展，否则利用substr函数返回当前的最长回文子串。

  4. 代码如下

string isPalding(string s,int left,int right){
    while(left>=0&&right<=s.size()-1&&s[left]==s[right]){
        left--;
        right++;
    }
    return s.substr(left+1,right-left-1);
}
string longestPalindrome(string s) {
    string res="";
    for(int i=0;i<s.size();i++){
        string s1=isPalding(s,i,i);
        string s2=isPalding(s,i,i+1);
        if(s1.size()>res.size()){
            res=s1;
        }
        if(s2.size()>res.size()){
            res=s2;
        }
    }
    return res;
}

时间复杂度是 O(n^2) ，空间复杂度是 O(1) ，其中 n 是字符串 s 的长度

1.大富翁游戏

题目

小美在玩大富翁游戏，游戏中有n+1个城市排成一排，编号从0到n，第i个城市有一个数字ai，表示到达第i个城市可以获得ai枚金币

每一轮开始时小美会获得四张卡牌，分别可以跳跃1、2、3、4个城市，例如小美可以从城市1跳跃3个城市到达城市4。当小美使用完这4张卡牌后，会开启新的一轮

初始时，小美拥有0枚金币，在任意时刻，小美的金币数量都必须大于等于0，小美想知道她从第0个城市出发，到达第n个城市最多可以获得多少枚金币

描述

输入描述

  1. 第一行输入一个整数n，表示城市个数
  2. 第二个输入n个整数，表示到达城市1到n可以获得的金币数量

输出描述

  1. 在一行上输出一个整数表示答案，如果无法到达第n个城市，则输出-1

示例

输入

10
-1 2 3 4 -9 -9 -1 3 -1 -1

输出

9

解释

  1. 第一步使用跳跃3的卡牌，从0跳到3，获得3枚金币
  2. 第二步使用跳跃1的卡牌，从3跳到4，获得4枚金币
  3. 第三步使用跳跃4的卡牌，从4跳到8，获得3枚金币
  4. 第四步使用跳跃2的卡牌，从8跳到10，获得-1枚金币
  5. 一共是9枚金币

思路

  1. 因为小美在每一轮中只能获得四张卡牌，分别跳跃1、2、3、4个城市，只有使用完这4张卡牌后，才会开启新的一轮
  2. 因此，可以以十个城市为一轮，小美每一轮的最后一定会到达第10个城市（或10个城市的整数倍）
  3. 当然，在最后一轮时，小美无需走完这一轮

解决

  1. 首先，利用回溯得到一个map<int,vector<vector<int>>>，其中【map的第i个二维数组】对应【行走到第i格的方法】，举例
步长4:3 1
步长4:4
步长5:1 4
步长5:2 3
步长5:3 2
步长5:4 1
步长6:1 2 3
步长6:1 3 2
步长6:2 1 3
步长6:2 3 1
  1. 在得到以上map后，基于动态规划定义一个【数组dp】，【dp[i]】为到达第i个城市时的最大金币数量
  2. 其实在每一轮中，对于任何步长的城市，都可以根据map计算其最大金币数量。举例，如果当前城市为15，则我们需要【dp[10]】，然后寻找map中步长为5的走法，分别为{1，4}、{2，3}、{3，2}、{4，1}，对这些走法依次计算金币数量，则可以得到【dp[15]】
  3. 代码如下：
#include <iostream>
#include <vector>
#include <map>
using namespace std;

// 得到map
void fun(map<int, vector<vector<int>>>& jump, vector<int> memory, vector<int> res, int step, int cur) {
    if (cur == step) {
        jump[step].push_back(res);
    }
    for (int i = 1; i <= min(step, 4); i++) {
        if (memory[i] == 0) {
            memory[i] = 1;
            res.push_back(i);
            fun(jump, memory, res, step, cur + i);
            res.pop_back();
            memory[i] = 0;
        }
    }
}

int main() {

    // 初始化城市数量和金币数量
    cout << "城市数量：";
    int n;
    cin >> n;
    cout << endl << "各个城市的金币数量：";
    vector<int> gold(n + 1);
    for (int i = 1; i <= n; i++) {
        cin >> gold[i];
    }

    // 创建10步以内的所有走法
    map<int, vector<vector<int>>> jump;
    for (int i = 1; i <= 10; i++) {
        vector<int> memory(5);
        vector<int> res;
        fun(jump, memory, res, i, 0);
    }

    // 定义一个数组dp，dp[i]表示到达第i个城市时的最大金币数量
    vector<int> dp(n + 1);

    // 定义一个结果存放走法
    vector<vector<int>> res;

    // 计算
    for (int i = 1; i <= n; i++) {
        // idx表示与上一轮的差
        int idx = i % 10;
        if (idx == 0) {
            idx = 10;
        }
        // nfront表示上一轮的终点
        int nfront = i - idx;

        // 计算金币数量
        // jump[idx]代表走idx的走法有多少种，计算这些走法的金币数量
        vector<int> tmp;
        for (int j = 0; j < jump[idx].size(); j++) {
            int num = dp[nfront];
            int step = 0;
            for (int t = 0; t < jump[idx][j].size(); t++) {
                step += jump[idx][j][t];
                num += gold[nfront + step];
            }
            // 更新dp
            if (num > dp[i]) {
                dp[i] = num;
                tmp = jump[idx][j];
            }
        }
        // 存放结果
        if (idx == 10 || i == n) {
            res.push_back(tmp);
        }
    }

    cout << endl << "金币数量：" << dp[n] << endl;
    cout << endl << "走法为：" << endl;
    for (int i = 0; i < res.size(); i++) {
        cout << "第" << i + 1 << "轮：";
        for (int j = 0; j < res[i].size(); j++) {
            cout << res[i][j] << " ";
        }
        cout << endl;
    }
    system("pause");
    return 0;
}

备注

以上题解没有经过案例通过，可能存在错误，请见谅

可能存在错误：

  1. 金币数量需要用【long long】定义，【int】定义可能会溢出
  2. 可以删除输出中的【走法】，题目只需要输出【最大金币数量】
  3. 没有进行【金币数量少于0】的判断，当i为10的整数倍且【dp[i]<0】时，无法到达目标城市，可以直接返回-1

2.剪彩带

题目

小美的彩带是由一条长度为n的彩带一直无限循环得到的，彩带的每一个位置都有一个颜色，用ai表示。因此当i>n时，ai=a(i-n)

小美每次会从左往后或从右往左剪一段长度为x的彩带，她想知道她每次剪下来的彩带有多少种颜色

描述

输入描述

  1. 第一行输入两个整数n，q代表彩带长度和剪彩带次数
  2. 第二行输入n个整数代表彩带每一个位置的颜色
  3. 此后q行，每行输入一个字符c和一个整数x，代表裁剪方向和裁剪长度，其中字符【’L’】说明从左往右剪，字符【’R’】说明从右往左剪

输出描述

  1. 对于每一次裁剪彩带，在一行上输出一个整数代表颜色数量

示例

输入

6 3
1 1 4 5 1 4
L 2
L 3
R 12

输出

1
3
3

解释

  1. 第一次剪彩带，剪下来的是【1，1】，只有1种颜色
  2. 第二次剪彩带，剪下来的是【4，5，1】，有3种颜色
  3. 第三次剪彩带，剪的长度超出彩带长度，包含彩带所有颜色，有3种颜色

思路

  1. 需要注意的是，在输入时，我们即可以记录【彩带包含的所有颜色数量sum】；且当剪彩带的长度超出彩带原长度时，可以直接返回【sum】，但是需要更新剪彩带的位置
  2. 首先，我们可以将彩带扩展为【长度为2n的两段一模一样的彩带】，然后定义一个指针【cur】指向上一次剪完彩带的位置
  3. 如果是从左往右剪，则将【cur】置于左段，然后往右判断剪下来的长度中有多少种颜色，从右往左剪同理，将【cur】置于右段

解决

代码如下

#include <iostream>
#include <vector>
#include <unordered_set>
using namespace std;

int fun(vector<int> color, int sum, int n, int& cur, char ch, int num) {
    // n表示彩带长度，cur表示当前彩带位置
    // ch表示左右，num表示剪彩带的长度
    if (ch == 'L') {
        cur %= n;
        unordered_set<int> memory;
        for (int i = cur; i < cur + num; i++) {
            memory.insert(color[i]);
        }
        cur += num;
        return memory.size();
    }
    else if (ch == 'R') {
        cur = cur % n + n;
        unordered_set<int> memory;
        for (int i = cur; i > cur - num; i--) {
            memory.insert(color[i]);
        }
        cur -= num;
        return memory.size();
    }
}

int main() {

    // 初始化
    // 第一行，彩带长度和剪彩带次数
    int n, q;
    cin >> n >> q;

    // 第二行，彩带每个位置的颜色，顺便记录彩带包含的所有颜色数量
    vector<int> color(2*n);
    unordered_set<int> memory;
    for (int i = 0; i < n; i++) {
        cin >> color[i];
        memory.insert(color[i]);
    }
    int sum = memory.size();

    // 拓展为两段彩带
    for (int i = n; i < 2 * n; i++) {
        color[i] = color[i - n];
    }
    // 初始位置为0
    int cur = 0;
    
    // 之后q行输入，并且记录结果
    vector<int> result;
    for (int i = 0; i < q; i++) {
        char ch;
        int num;
        cin >> ch >> num;
        if (num >= n) {
            result.push_back(sum);
            cur = ch == 'L' ? (cur + sum) % n : (cur - sum) % n;
        }
        else {
            int res = fun(color, sum, n, cur, ch, num);
            result.push_back(res);
        }
    }

    // 输出
    for (int i = 0; i < q; i++) {
        cout << result[i] << endl;
    }

    system("pause");
    return 0;
}
